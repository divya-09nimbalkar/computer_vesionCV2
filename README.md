````markdown
# Car Detection using OpenCV

## Overview
This project detects cars in images and video streams using OpenCV (CV2) and Python. Detected vehicles are highlighted with bounding boxes.

## Technologies Used
- Python
- OpenCV (CV2)
- NumPy

## Features
- Real-time car detection
- Video and image processing
- Bounding box detection

## Installation
```bash
pip install opencv-python numpy
````

## Run the Project

```bash
python car_detection.py
```

## Applications

* Traffic monitoring
* Vehicle counting
* Smart parking systems

